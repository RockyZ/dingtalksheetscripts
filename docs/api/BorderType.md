# BorderType

## API描述

边框类型。用于 Range.setBoder() API中指定要操作的边框类型。

## 属性说明

| 属性 | 类型 | 是否必填 | 说明 |
| --- | --- | --- | --- |
| top | boolean | 否 | 顶部位置。 |
| bottom | boolean | 否 | 底部位置。 |
| left | boolean | 否 | 左边位置。 |
| right | boolean | 否 | 右边位置。 |
| vertical | boolean | 否 | 垂直位置（除去两边）。 |
| horizontal | boolean | 否 | 水平位置（除去两边）。 |
